# TraceBack

A VS Code extension that brings telemetry data (traces, logs, and metrics) into your code.

## Demo

![Traceback 0 5 Boomerang demo](https://github.com/user-attachments/assets/6fb626fa-84da-4e62-9963-64f97d9a80bf)
